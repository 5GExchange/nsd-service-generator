#### Usage
Set the following parameters in the script according to the ones in the VNFD(s) (domain, provider_id and vnf_port_id), the SAPs in the low level infrastructure and the desired number of nf instances:

```
domain = 'UCL'
provider_id = '4'
ingress = 'SAP0:in'
egress = 'SAP1:out'
vnf_port_id = '99'
nf_instances = 100
```
